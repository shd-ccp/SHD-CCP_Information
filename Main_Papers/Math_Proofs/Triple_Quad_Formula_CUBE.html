Triple Quad Formula
for a 4×4×4 Cube

Derivation extending the Triple Quad Formula from 1D affine space to 3D affine space, applied to a discrete lattice grid.

independent.academia.edu/GHoard

Statement

The Triple Quad Formula (TQF) asserts that for three collinear points, the Archimedes function of their quadrances is zero. The goal is to extend this relation from 1D affine space into 3D affine space — specifically applied to a discrete 4×4×4 lattice grid.

The Triple Quad Formula (Q₁ + Q₂ + Q₃)² = 2(Q₁² + Q₂² + Q₃²)
Step 0 The 3D Derivation
Defining three collinear points in Euclidean space

Define three points A₁, A₂, A₃ in 3D Euclidean space. If collinear, they lie on a single line that can be parameterized with a direction vector v = (dx, dy, dz).

A₁
(x₁, y₁, z₁)
A₂
(x₂, y₂, z₂)
A₃
(x₃, y₃, z₃)
Let A₂ = A₁ + v
And A₃ = A₁ + kv

where k is a scalar representing the distance ratio along the line.

Step A Define "Quadrance"
Rational Trigonometry's squared-distance measure
Quadrance (Q)

In Rational Trigonometry, we avoid square roots to keep the math rational. Instead of calculating distance — which often results in irrational numbers — we calculate Quadrance, which is the square of the distance. Magnitude is expressed not by how long a line is, but by the area of the square that line creates.

Q(A, B) = (x_b − x_a)² + (y_b − y_a)² + (z_b − z_a)²

First, calculate the Base Quadrance of vector v:

Qv = dx² + dy² + dz²

Now calculate the three interaction quadrances:

Quadrance Points Expression
Q₁ A₁ → A₂ 1² · Qv = Qv
Q₂ A₂ → A₃ (k − 1)² · Qv
Q₃ A₁ → A₃ k² · Qv
Step B Substitute into TQF
Expanding LHS and RHS with the interaction quadrances

We substitute Q₁, Q₂, Q₃ into both sides of the theorem. Note that Qv is a common factor; we simplify by setting Qv = 1 for the algebraic proof since the formula is homogeneous.

LHS
(Q₁ + Q₂ + Q₃)²
= (1 + (k−1)² + k²)²
Expand
= (1 + k² − 2k + 1 + k²)²
= (2k² − 2k + 2)²
Factor
= 4(k² − k + 1)²
— — —
RHS
2(Q₁² + Q₂² + Q₃²)
= 2(1 + (k−1)⁴ + k⁴)
Expand
= 2(1 + k⁴ − 4k³ + 6k² − 4k + 1 + k⁴)
Simplify
= 4k⁴ − 8k³ + 12k² − 8k + 4
= 4(k⁴ − 2k³ + 3k² − 2k + 1)
Factor
= 4(k² − k + 1)²
Conclusion
Since LHS = RHS, the Triple Quad Formula holds for any 3 collinear points in 3D space.
LHS = 4(k² − k + 1)² = RHS ✓
Application to the 4×4×4 Cube
SHD-CCP 64-bit packet — discrete lattice grid

The 4×4×4 lattice contains 64 discrete nodes with integer coordinates in {0,1,2,3}³. The TQF applies to any collinear triple within this structure, enabling rational distance comparisons without irrational square roots.
